Clase: AppLogger

Herencia: Hereda de Logger

La clase AppLogger es una implementación de puerto (Logger port) que utiliza la librería estándar logging de Python para gestionar la salida de registros de la aplicación.

Constructor

__init__(self, name: str) -> None

Inicializa una nueva instancia del logger.

  • Parámetros:
    • name (str): El nombre del logger, usualmente utilizado para identificar el origen de los mensajes (ej. __name__).
  • Descripción: Crea una instancia interna de logging.Logger obtenida mediante logging.getLogger(name).

Métodos Públicos

info(self, message: str) -> None

Registra un mensaje con nivel de severidad INFO.

  • message: El contenido del texto a registrar.

debug(self, message: str) -> None

Registra un mensaje con nivel de severidad DEBUG.

  • message: Detalles técnicos útiles para la depuración.

warning(self, message: str) -> None

Registra un mensaje con nivel de severidad WARNING.

  • message: Indicación de que algo inesperado sucedió o hay un problema potencial.

error(self, message: str) -> None

Registra un mensaje con nivel de severidad ERROR.

  • message: Informe de un problema serio que impidió la ejecución de una función específica.

Atributos Privados

  • self._logger: Instancia interna de la librería estándar de Python encargada de la lógica de registro.
